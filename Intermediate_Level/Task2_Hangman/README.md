# Task 2: Hangman

This is a Python implementation of the classic Hangman word-guessing game.

## Features

- **Visual Progress:** Displays an ASCII art representation of the hangman as you make incorrect guesses.
- **Hints:** Provides a hint at the beginning of the game to help you guess the secret word.
- **Replayability:** The game allows you to play multiple times in a row without restarting the script.
- **Tracking:** Keeps track of the letters you have already guessed to prevent duplicate penalties.

## How to Play

1. Run the script:
   ```bash
   python hangman.py
   ```
2. Read the hint provided at the start of the game.
3. Guess the word by typing one letter at a time and pressing Enter.
4. Try to guess the complete word before the hangman is fully drawn!
